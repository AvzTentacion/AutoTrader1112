IMPORTANT - READ BEFORE OPENING IN ANDROID STUDIO
==================================================

1. Copy YOUR original gradlew and gradlew.bat files into the root of this folder.
2. Copy gradle/wrapper/gradle-wrapper.jar from your original project.
3. Copy your drawable resources (auto.png and userlogo.png) into:
   app/src/main/res/drawable/

4. Open the AutoTrader folder in Android Studio:
   File > Open > select the AutoTrader folder
